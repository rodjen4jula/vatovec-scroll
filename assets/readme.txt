Adds a scroll to top and bottom which allows the visitors to easily scroll back to the top or the bottom of the page.
  
== Description ==  

This plugin adds a custom Back to Top and Back to Bottom buttons to your WordPress website that allows visitors to scroll back to the top of your page with one click.
  
== Installation ==
  
1. Upload the plugin folder to your /wp-content/plugins/ folder.
1. Go to the **Plugins** page and activate the plugin.
  
== Frequently Asked Questions ==
  
= How do I use this plugin? =
  
After you activate the plugin you can clik on the settings link or go to the Vatovec Scroll page in side bar. There you on main page tab you can activate scroll to top and scroll to bottom also after acitvatin one or both options you can activate the option show on admin if you want to have scroll on admon page.
  
= How to uninstall the plugin? =
  
Simply deactivate and delete the plugin.
